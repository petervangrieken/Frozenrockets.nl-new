The new face of Frozen Rockets

I'm proud to finally present the new identity and website of Frozen Rockets.

A few months ago I asked my friend and awesome designer Stephan Csikós to work on a new brand identity for me. I'm very pleased with the result he came up with. Seriously, if you need a great graphic designer or art director, hire him!

Based on his work, I designed this website, where I 'practice what I preach', so of course it's completely keyboard-accessible and it conforms to WCAG 2.0 (accessibility) standards without it being 'boring'.

What's next?

I'll go into more details about some of the design choices and implementations on this site in more detail in some follow-up articles later, but if you like to look around, feel free to check out the source code on GitHub. Also, if you run into any issues, please let me know.

The great thing about the web is that it can be changed at any time, so I deliberately left some work to be done (like adding ARIA roles), which I'll cover in upcoming articles on this site.
