Responsive design

De verscheidenheid aan apparaten waarmee we tegenwoordig het web op kunnen is enorm. Het is dan ook ondenkbaar dat een website er alleen maar goed uit ziet op een 21" desktopscherm. Een goed Responsive design zorgt voor een website die blijft werken op alle apparaten.

Bij responsive design wordt in de eerste plaats vaak gedacht aan CSS en media queries, maar begint bij een goed proces waarin content en nadenken over prioriteiten centraal staan. Een mobile first aanpak kan hier bij helpen. Door vanaf het begin uit te gaan van kleine schermen en van daaruit uit te breiden, worden alle partijen gedwongen om keuzes te maken. Zowel in de ontwerpfase bij het bepalen van de content en functionaliteit, als bij de implementatie om de CSS compact en goed onderhoudbaar te houden.

Ik heb veel ervaring bij het ontwerpen en implementeren van responsive websites, zowel voor grote als kleine websites. Hoewel dit voor elk project geldt, wordt door responsive design nog meer benadrukt hoe belangrijk een goede samenwerking tussen alle disciplines is.
