Frontend development

De basis van webdesign is HTML, de structuur die het fundament vormt van uw website. Goed gebruik van webstandaarden heeft niet alleen invloed op de toegankelijkheid van uw website voor screenreaders, maar ook op de laadsnelheid van de pagina's en op uw positie in zoekmachines.

HTML en CSS zijn twee van de meest toegankelijke talen om te schrijven. Met een eenvoudige text-editor en webbrowser kan iedereen snel beginnen met het maken van een website, toch wordt van goed frontend development meer verwacht.

In de 25 jaar dat het web nu bestaat zijn de eisen die aan websites gesteld worden enorm veranderd. Van het moment dat CSS nog maar een idee was tot nu, waarin CSS is uitgegroeid tot een volwaardige designtool die vraagt om schaalbare oplossingen.

Ik ben gespecialiseerd in goed onderhoudbare HTML en CSS die zorgt voor snelle en toegankelijke websites. Ik vind het belangrijk om een goede frontend neer te zetten die werkt in het brede scala van hedendaagse browsers en apparaten.

Mijn voorkeur voor toegankelijke websites zorgt dat ik hoge eisen stel aan mijn frontend code op dit vlak. De juiste keuze voor buttons en links en de juiste volgorde van koppen, zorgen niet alleen voor nette code, maar ook voor een website die beter gebruikt kan worden door bezoekers met een beperking.
