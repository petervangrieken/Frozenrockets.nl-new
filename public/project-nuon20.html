Een nieuwe website voor Nuon

In 2013 heb ik voor Nuon de frontend voor het vernieuwde Nuon.nl verzorgd. Naast een nieuw CMS, een vernieuwde backend voor verschillende tools en een grote contentmigratie, was er een grote update van de frontend nodig. Het design was verouderd en ook de CSS kon door een aantal jaren legacy een opfrisbeurt gebruiken.

Good designers redesign, great designers realign.
Cameron Moll

Op basis van de markup die al grotendeels bepaald was, is een nieuwe ontwerp geïmplementeerd. Hierbij was een van de uitgangspunten dat het ontwerp zo dicht mogelijk bij het oude onwerp moest blijven. Het nieuwe ontwerp moest voornamelijk zorgen voor een moderner uiterlijk en meer consistentie.

In het oude ontwerp werd veel gebruik gemaakt van afgeronde hoeken en schaduwen. Deze zijn voor een groot deel weggelaten in het nieuwe onwerp, waardoor de site niet alleen moderner aanvoelt, maar ook sneller geworden is.

Beter voorbereid op de toekomst

De CSS is volledig opnieuw geschreven, gebaseerd op OOCSS, wat zorgt voor herbruikbare code. Dit heeft geresulteerd in een CSS die aanzienlijk (~50%) kleiner is geworden. Daarnaast is gebruik gemaakt van CSS in plaats van Javascript waar mogelijk wat voor een verdere optimalisatie in de laadtijd gezorgd heeft.

Het project heeft door deze en andere wijzigingen voor een verbetering gezorgd voor de laadtijd van de site, de toegankelijkheid en de algehele gebruikerservaring. Dit heeft ook een positieve invloed gehad op de SEO en de conversie van de site.

De OOCSS opzet zorgt dat de site beter voorbereid is op de toekomst, het is makkelijker geworden om wijzigingen op het design door te voeren en door goed hergebruik van styles blijft de consistentie behouden.

Pattern Library

Als onderdeel van het project heb ik een Pattern Library opgeleverd waarin alle componenten die op de site gebruikt worden gedocumenteerd staan met de bijbehorende HTML. Deze library kan intern gebruikt worden als documentatie voor developers en contentmanagers, maar ook door externe leveranciers als naslagwerk bij het bouwen van tools.
